# Screen Studio Polish - Git Weekly Report

The automated VHS clips are readable and deterministic. Use Screen Studio only
when you want smoother camera motion or a cursor-polished export.

## Source Video

```text
demo/pipeline/output/gitweeklytracker-demo.mp4
```

Create a project from that MP4, set the aspect ratio to 16:9, and export at
1920x1080, 30fps.

## Manual Zoom Beats

- Scope clip: keep wide so the environment variables and repo log are both
  visible.
- Generate clip: slow zoom into the summary section after the report renders.
- Markdown clip: zoom slightly into the table and summary heading.
- Evidence clip: hold on `Top Changed Files`, then the daily activity table.

Keep the total length near 30-35 seconds.
